# Contract Dashboard 3D Assets

These local assets are selected from Microsoft Fluent Emoji 3D.

- Source: https://github.com/microsoft/fluentui-emoji
- License: MIT
- Usage: local UI illustration assets for the Contract Journey dashboard; no CDN dependency.

File mapping:

- `ship-loading.png`: Passenger ship
- `storage-tank.png`: Oil drum, used as the closest storage/tank metaphor in the same 3D family
- `fuel-pump.png`: Fuel pump
- `tank-truck.png`: Delivery truck, used for truck/tank transport
- `customs-bank.png`: Classical building, used for customs
- `warehouse-receipt.png`: Bank, used as warehouse/terminal receipt metaphor
- `shop-sale.png`: Department store, used for sales
- `coins-profit.png`: Money bag, used for profit
- `expense-donut.png`: Bar chart, used for cost breakdown
- `loss-warning.png`: Warning
