Design PTG UI cleanly and minimally.

Rules:
- UI must be clear, simple, and not crowded.
- Do not add too many tabs.
- Maximum 3-5 main sections.
- Keep Bootstrap 5 RTL.
- Keep existing ptg-* design tokens.
- Do not break other pages.
- Remove conflicting old styles only when proven.
- Prefer simple cards, readable tables, clear money summaries.
- Do not redesign unrelated pages.
